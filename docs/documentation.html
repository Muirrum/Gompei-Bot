ban

Admin

Bans a user from the server.

.ban <user>

clearWarn

Mod

Clears the warnings for a user.

.clearWarn <user>

kick

Admin

Kicks a user from the server.

.kick <user>

lockdown

Mod

Locks down a channel so anyone without explicit send message permissions cannot send messages.

.lockdown

.lockdown <channel>

mute

Mod

Gives the user the designated muted role for the amount of time given. Also DMs the reason for the mute.

.mute <user> <time> <reason>

purge

Mod

Purges messages from a channel.

.purge <number>

slowmode

Mod

Sets the slowmode for a channel. Maximum slowmode is 6 hours.

.slowmode <channel> <time>

spurge

Mod

Purges messages from a specific user - "specific purge".

.spurge <user> <number>

unlock

Mod

Unlocks a channel that was in lockdown.

.unlock

.unlock <channel>

warn

Mod

Adds a warn to a user and DMs them the warn.

.warn <user>

warnNote

Mod

Adds a warn to a user without sending the warn to them.

.warnNote <user>

warns

Mod

Shows all of the warnings made in the server or warnings for a specific user.

.warns

.warns <user>

addAccessRole

Admin

Adds role(s) to the access role list. Access roles are roles that are used to get access to normal server channels.

.addAccessRole <role(s)>

.addAccessRole clear

addOptInRole

Admin

Adds role(s) to the opt-in role list. Opt-in roles are roles that are used to get access to specific server channels.

.addOptInRoles <role(s)>

.addOptInRoles clear

letmein

Returns all roles taken away from a lockout command.

.letmein

lockout

Removes all roles from a user to later be returned by letmein.

.lockout

prefix

Admin

Sets the prefix for the bot.

.prefix <newPrefix>

removeAccessRole

Admin

Removes access role(s) for the server. Access roles are roles that are used to get access to normal server channels.

.removeAccessRole <roles>

.removeAccessRole clear

removeOptInRole

Admin

Removes opt-in role(s) for the server. Opt-in roles are roles that are used to get access to specific server channels.

.removeOptInRole <roles>

.removeOptInRole clear

rradd

Mod

Adds a reaction role to a message.

.rradd <messageLink> <role> <emoji>

rremove

Mod

Removes a reaction role from a message.

.rremove <messageLink> <emoji>

status

Mod

Changes the bots status to "Playing <status>".

.status <status>

logging

Admin

Sets the logging channel for the server.

.logging <channel>

hangman

Creates a reaction hangman game.

.hangman

minesweeper

Creates a minesweeper game.

.minesweeper

.minesweeper <x> <y>

addMovie

Adds a movie to the movie vote.

.addMovie <movieName>

listMovies

Sends the current movie voting leaderboard.

.listMovies

movieInfo

Sends IMDB info for the movie.

.movieInfo <movieName>

removeMovie

Admin

Removes a movie from the vote.

.removeMovie <movieName>

removeVote

Removes you vote made for a movie.

.removeVote <movieName>

resetMovies

Admin

Resets the movie vote.

.resetMovies
